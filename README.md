## Installation 
1.	Clone the repository: 
git clone https://github.com/shtwaijri/Educational-Kids-Game.git

2.	Navigate to the project folder: 
cd  Educational-Kids-Game

3. Install dependencies: 
flutter pub get 

4.	Run the app: 
flutter run
